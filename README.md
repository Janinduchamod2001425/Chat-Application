# Chat-Application

### create React + vite frontend

```
npm create vite@latest
```

### run the frontend folder

```
cd frontend
npm install
npm run dev
```

### run the backend folder

```
cd backend
npm install
npm run dev
```

### install node packages to the backend folder

```
npm init -y
```

### install nodemon devDependencies

```
npm i nodemon -D
```
